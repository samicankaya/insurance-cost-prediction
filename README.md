#  Health Insurance Cost Prediction Model

This project utilizes Machine Learning (Linear Regression) to predict annual medical insurance costs based on individual characteristics such as age, gender, BMI, and smoking status.

##  Features
- **Data Analysis:** Data visualization using Pandas and Seaborn.
- **Model:** Built with Scikit-Learn Linear Regression.
- **Random Scenario:** Generates a random customer profile on every execution to provide dynamic price predictions.

##  Installation & Usage

1. Clone the repository:
   ```bash
   git clone [https://github.com/samicankaya/insurance-cost-prediction.git](https://github.com/samicankaya/insurance-cost-prediction.git)

2. Install the required libraries:
      ```bash
   pip install -r requirements.txt

3. Run the application:
   ```bash
   python main.py
